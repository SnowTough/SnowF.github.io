Docker简介

Docker 是什么

​ Docker 是一个应用打包、分发、部署的工具

​ 可以把它理解为一个轻量的虚拟机，它只虚拟你的软件需要的运行环境

​ 而普通的虚拟机则是一个完整而庞大的系统，包含各种软件，无论是否需要

打包、分发、部署

​ 打包：把软件运行所需要的依赖、第三方库、软件打包到一起，变成一个安装包

​ 分发：把打包好的“安装包”上传到一个镜像仓库，其他人可以非常方便地获取和安装

​ 部署：得到这个“安装包”，使用一个命令运行起你的应用，自动模拟出相同的运行环境，无论是在什么系统下

Docker 一般用来做什么？

  • 应用分发、部署，方便传播给他人安装。特别是开源软件和提供私有部署的应用
  • 快速安装测试/学习软件，用完即丢（类似小程序），不把时间浪费在安装软件上。例如 Redis / MongoDB / ElasticSearch / ELK
  • 多个版本软件共存，不污染系统，例如：Python2, Python3, Redis4.0, Redis5.0
  • Windows 和体验各种 Linux 系统

（Important）镜像、容器

​ 镜像：可以理解为软件安装包，可以方便地进行传播和安装

​ 容器：软件安装后的状态，每个软件运行环境都是独立的、隔离的，称之为容器
